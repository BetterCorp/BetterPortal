# BetterPortal v10 Overview

BetterPortal v10 is organized by product category first and language second:

- `framework/nodejs`
- `auth/nodejs`
- `themes/nodejs/bootstrap1`
- `themes/nodejs/embedded`
- `services/nodejs/examples/hello-view`

Core principles:

- API-first views
- HTML as negotiated representation
- HTMX-first UI composition
- auth and theme as plugin categories
- strict type safety
