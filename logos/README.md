# BetterPortal Logo - Final Candidate A

Selected direction: clean modular core.

Concept:
Four service/module tiles surrounding a central platform core. It keeps the simplicity of the modular option without leaning too hard into Windows-style pane branding.

Palette:
- Base: #020617
- Blue: #2563EB
- Cyan: #22C7D9 / #38BDF8
- Green: #65C832
- White: #F8FAFC

Included:
- betterportal-logo.svg - transparent source icon
- betterportal-logo.png - 256x256 transparent PNG
- betterportal-logo@512.png
- betterportal-logo@1024.png
- betterportal-logo@128.png
- betterportal-favicon-64.png
- betterportal-favicon-32.png
- betterportal-favicon-16.png
- betterportal-logo-light.svg/png
- betterportal-logo-dark.svg/png
- betterportal-logo-mono.svg/png
- betterportal-wordmark.svg/png

Suggested usage:
- Use betterportal-logo.svg for docs, web, and UI branding.
- Use betterportal-favicon-32.png for favicon fallback.
- Use betterportal-logo@512.png for package/plugin images. The BSB registry upload path accepts PNG assets, not SVG.
